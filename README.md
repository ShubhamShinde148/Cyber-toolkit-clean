# Cyber Security Toolkit

A modular full-stack cybersecurity toolkit built with Express + vanilla frontend.

## Run locally

```bash
npm install
npm start
```

Open: http://localhost:3000

## Deploy on Railway

1. Push this folder to a GitHub repository.
2. In Railway, create a new project from that repo.
3. Railway will auto-detect `package.json` and run `npm install`.
4. Start command: `npm start` (already set in `railway.json`).
5. After deploy, open `/health` to verify service status.
